# Chef's Recipe Journal – Developer Website

This repository hosts the developer website files for **Chef's Recipe Journal**, an Android application available on the Google Play Store.

## Purpose

The main purpose of this repository is to host the **app-ads.txt** file required for advertising verification through Google AdMob. This helps ensure authorized ad inventory and prevents ad fraud.

## App Information

**App Name:** Chef's Recipe Journal
**Platform:** Android
**Package Name:** com.gerwin.chefsrecipejournal

Chef's Recipe Journal is a digital kitchen companion that allows users to scan, digitize, organize, and share their recipes.

## app-ads.txt

The `app-ads.txt` file in this repository verifies that **Google AdMob** is an authorized seller of ad inventory for the app.

Content of `app-ads.txt`:

google.com, pub-3483365512758140, DIRECT, f08c47fec0942fa0

## Google Play Listing

The app is published on the Google Play Store and uses this repository's GitHub Pages site as the developer website for advertising verification.
